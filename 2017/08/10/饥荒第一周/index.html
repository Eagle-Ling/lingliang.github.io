饥荒第一周

发表于 2017-08-10 | 分类于 生活杂记 | | 阅读次数


要说我为啥这么喜欢饥荒，原因有三：

  1. 画风荒诞奇特，我很喜欢。
  2. 我以前就很喜欢看荒野求生的节目，饥荒带给我的游戏体验让我身临其境。
  3. 现实中不能做的事情，在游戏中我都能做哈哈哈。

    哦！还有最重要的一点，就是每次死得都很不甘心，饥荒虐我千百遍，我仍待它如初恋QAQ…

现在放暑假，每天也挺无聊的，为了留下点生活的痕迹，我每天玩了饥荒之后都会画点什么来记录一下当天的饥荒生活，以后就按周发饥荒周记吧，反正我也不是天天都那么勤奋的去画。

周一：“领地争夺战”


这是一场足以载入史册的领地争夺战！！战况之激烈，所到之处，怪兽肉遍地，猪皮成河！
最后以威尔逊全端了蜘蛛老巢的胜利而告终。
此役我的猪人们英勇顽强，不畏强敌，浴血奋战！以牺牲了7个猪人战士的惨痛代价，取得了全面的胜利！
仅以此图，祭奠我7位英勇牺牲的猪人战士！以告烈士之灵。
R.I.P

周二：“菜地里的奇怪东西”


不知道为啥，菜地里突然长出了一颗奇怪的东西，然后周围还有恶心的眼珠子，被逼出密集恐惧症，也不知道好吃不好吃 - -||

周三：“该死的火鸡”


开始我是很讨厌火鸡这东西的，天天在我的浆果丛里打秋风，还特么跑的贼快，窃取我的劳动成果。所以说做人不能太“火鸡”。

周四：“我爱火鸡”


经过Same上大神们的指点，笑死。。。火鸡简直就是上帝派到人间送温暖的天使哈哈哈哈。回旋镖怕不怕，是你跑得快还是我的镖快！还有你不是很能吃吗？地上放两个红蘑菇，这货果然不挑食，然后吃完就GG了，所以晚上吃了一波火鸡大餐，顺便犒劳一下我的猪人兄弟~~

周五：“最遥远的距离”


世界上最遥远的距离不是生与死，而是我偷了一个蛋，你追我从天涯追到海角。。。

周六：“吃瓜群众”


这天正当我被这头蠢鹿追的无路可逃时，前面突然出现了一只超级大苍蝇，然后它俩就打起来了，我默默拿出我的西瓜，为它们呐喊助威，真是山穷水复疑无路，柳暗花明又一村啊。O(∩_∩)O

周日：“脚印”


某人说顺着脚印找会有惊喜，我有一句mmp一定要讲！！ #@￥%&*

这破游戏很完美的诠释了一句话————生活处处有惊喜
