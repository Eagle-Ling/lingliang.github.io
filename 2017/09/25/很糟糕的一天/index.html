非常糟糕的一天

发表于 2017-09-25 | 分类于 生活杂记 | | 阅读次数

      今天，我失去了一个朋友，她叫陈修哲。

      我们是在今年暑假一个叫守望先锋里的游戏里认识的。在一次机缘巧合的快速游戏里，她玩的天使，我玩的安娜，我们俩玩的都是辅助，开始我不知道啊为啥她加我好友，后来才了解到那局是因为我发挥得很不错，让她留下了深刻的印象。后来我们就开始一起开黑，虽然一直掉分，但是玩得还算很开心，毕竟她是我玩游戏认识的第一个小姐姐，听声音应该是一个很可爱的女孩子。

      后来我们加了微信，慢慢地从游戏聊到生活的琐事，虽然她比我大几岁，但是并没有任何代沟，我很感谢命运的安排。因为我这个人生性孤僻，不合群，知心的朋友也就那么几个，都是从各种矛盾中吵吵闹闹过来的，认识新的朋友对我来说很不容易。直到现在，我都是怀着非常难过的心情在写，我希望这能让我好一些，可是为什么越写越想哭呢。

      唉…继续写，反正这些话没人可道，这些文字也没人会看，倾诉出来了，心情就会好多的吧。修哲玩的最多的是天使，因为我没见过修哲的照片，所以很容易就把她想象成像天使那样的女孩子，心地善良，待人友好（在后来的每次聊天中我觉得确实如此）。我玩安娜时经常会被队友忽视，容易被敌方切，可有修哲的天使在身边，我每次被切她都在帮我，慢慢地，我对修哲产生了一点好感。嗯，很纯粹的友情+一点儿好感，我觉得这是很正常的感情，并没有什么苟且龌龊之处。

      可是一起和修哲打排位打多了，我就发现，我的水平渐渐跟不上修哲了，没错，我在拖累她，虽然她不会这么想，但比赛输多了，谁也会不开心，而我，绝对比她更不开心，因为不仅输了比赛，我还要承担因为拖累她而产生的内疚感。于是我诞生了一个幼稚且愚蠢的想法，自己单排上分，等自己打到钻石段位了，就说明自己不会再拖累修哲了，然后就可以带她一起上分了。但是这很难，今天我一直掉分，而且她也一直掉分，然后她想和我一起玩饥荒，然而该死的服务器启动不了，我本来心情就很差了，然后一气之下向修哲表达出烦躁而且不想再和她玩游戏的想法。可我们就是通过游戏相识的啊，我能理解，在她的角度来说，本来就是一个通过游戏而认识的朋友，没有了游戏的羁绊，这样的朋友就没有意义了，于是她把我删了，一切的社交关系。

      可我已经不单纯把她当作游戏里的朋友了，同时还是生活上的朋友，虽然我们素未谋面，但已经产生过许多精神的共鸣，在某些方面是很投机的。对于我，认识新朋友本已不易，更多的是生命中的匆匆过客，认识聊得来的朋友就更不容易了！落花有意随流水，流水无心恋落花。我知道，今晚过错在我，我要努力挽回这段友情！可是，修哲的态度，看起来是不太可能给我机会的了。。。

      唉，很难受，今晚注定又是一无眠之夜，不想再写了，如果上苍有眼，让这段缘分不要收之草草。请原谅我，修哲。
