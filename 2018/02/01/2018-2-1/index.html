《2.5D插画小练习》

发表于 2018-02-01 | 分类于 UI | | 阅读次数

练习对象

工具

  • PS：用来上色以及打造细节
  • AI：制作基础的形体和框架

    Tips

  • 在用AI的3D效果制作出理想形状时，扩展外观可以把整体拆分为局部，方便在PS里上色及处理，但扩展外观后3D效果的数值将不能在外观面板中更改。
  • 确保AI里做了3D效果的图形全部都扩展外观了后。直接选择工具选中所有图形后按Ctrl+C，跳转到PS中后按Ctrl+V，这时会有个对话框，选择形状图层即可。然后开始进行最考验耐心的一步啦路径选择工具选中每一个小块后按Shift+Ctrl+J即可将每一小块作为一个单独的图层提取出来。
  • 填色颜色时，如果用在属性栏中填充渐变，则可在用剪切蒙版覆盖，如果用图形样式里的渐变填充，则上面的剪切蒙版不会显示效果。在蒙版中添加杂色的方法，蒙版上可以添加多次杂色。
  • 在扩展形状的时候，一定要注意图形明暗交界线分布和组合图形之间的边缘衔接，如果边缘衔接和明暗交界线不很吻合，那么后期填充渐变颜色时很难达到理想效果。

总结

最终效果：


  1. 构图方面：各种几何体的透视关系
  2. 光影效果：高光、明暗交界线、阴影的分布（即渐变的运用）。
  3. 颜色：该风格使用颜色为饱和度偏低，名度高的颜色。
  4. 质感：用添加杂色的方法制造噪点，用高斯模糊+剪切蒙版制造阴影。
